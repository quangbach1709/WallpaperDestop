# WallpaperDestop 🖼️

**WallpaperDestop** là một ứng dụng Windows (WPF) mạnh mẽ và nhẹ nhàng, giúp biến không gian làm việc của bạn trở nên sống động bằng cách tự động tải và cài đặt những hình nền chất lượng cao nhất từ **Unsplash**.

## ✨ Tính năng nổi bật

- **Tự động đổi hình nền:** Cập nhật hình nền máy tính và màn hình khóa tự động từ kho ảnh khổng lồ của Unsplash.
- **Tìm kiếm theo chủ đề:** Cho phép cấu hình các từ khóa ảnh yêu thích (Thiên nhiên, Kiến trúc, Tối giản...).
- **Lịch sử hình nền:** Lưu giữ danh sách các ảnh đã từng sử dụng để bạn có thể xem lại hoặc tải về bất cứ lúc nào.
- **Chạy cùng hệ thống:** Tùy chọn tự khởi động khi mở máy để bạn luôn có hình nền mới mỗi ngày.
- **Thông báo thông minh:** Hiển thị thông báo khi hình nền được cập nhật thành công.
- **Tùy chỉnh linh hoạt:** Thay đổi thời gian cập nhật, chất lượng hình ảnh và nhiều cài đặt khác.

## 🚀 Hướng dẫn cài đặt

### Yêu cầu hệ thống
- Hệ điều hành: Windows 10 hoặc Windows 11.
- .NET Framework hoặc .NET Core (tùy thuộc vào phiên bản build).

### Cách sử dụng
1. **Tải về:** Clone repository này về máy:
   ```bash
   git clone https://github.com/quangbach1709/WallpaperDestop.git
   ```
2. **Mở dự án:** Sử dụng Visual Studio 2022 hoặc JetBrains Rider để mở tệp `WallpaperDestop.sln`.
3. **Cấu hình API (Nếu cần):** Đảm bảo bạn có Access Key từ [Unsplash Developer Portal](https://unsplash.com/developers) nếu phiên bản yêu cầu key cá nhân.
4. **Build & Run:** Nhấn `F5` để biên dịch và bắt đầu sử dụng.

## 🛠️ Công nghệ sử dụng

- **WPF (Windows Presentation Foundation):** Giao diện người dùng hiện đại.
- **MVVM Pattern:** Kiến trúc mã nguồn sạch, dễ bảo trì.
- **Unsplash API:** Nguồn cung cấp hình ảnh chất lượng cao.
- **Newtonsoft.Json:** Xử lý dữ liệu cấu hình và API.
